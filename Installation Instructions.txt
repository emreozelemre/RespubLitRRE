GENERAL REQUIREMENTS (ALL PLATFORMS)

Create a free Hugging Face account:

https://huggingface.co/emreozelemre/RespubLitRRE-LoRA

===========================================================
WINDOWS INSTALLATION (PowerShell)

Install Python from:
https://www.python.org/downloads/windows/

Make sure to tick "Add Python to PATH" during installation.

Install Git from:
https://git-scm.com/download/win

Open PowerShell:
Start Menu -> type "PowerShell" -> Enter.

Download the application:
git clone https://github.com/emreozelemre/RespubLitRRE.git

cd RespubLitRRE

Create and activate a virtual environment:
python -m venv .venv
..venv\Scripts\Activate.ps1

Install required packages:
pip install --upgrade pip
pip install -r requirements.txt

Log into Hugging Face:
huggingface-cli login

Run the application:
python RespubLitRRE_Gradio.py

Open the browser and go to:
http://localhost:7860

MACOS INSTALLATION (Terminal)

Install Homebrew (if not installed):
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh
)"

Install Python and Git:
brew install python git git-lfs
git lfs install

Download the application:
git clone https://github.com/emreozelemre/RespubLitRRE.git

cd RespubLitRRE

Create and activate the virtual environment:
